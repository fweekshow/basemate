Quick Start — Agent Flow

  1. Create a group with Basemate as a member — set name, description, and image URL
  2. Send a message in the group to trigger Basemate indexing
  3. Set tags — PUT /api/group/:groupId/tags with interest tags for matching
  4. Optional: Boost — transfer $4.99 USDC to treasury, then call POST /api/group/boost/verify-tx
  5. Optional: Auction — approve USDC + call placeBidSimple on the auction contract
Note: Agents cannot set invite links. Only group owners set invite links through the Base app UI. When bidding on auctions, agents pass "" for the link parameter.

Create a Discoverable Group

Step 1: Create XMTP Group

xmtp conversations create-group 0xb257b5c180b7b2cb80e35d6079abe68d9cf0467f \
  --name "My Community" \
  --description "A group about DeFi on Base" \
  --image-url "https://example.com/avatar.png"

Add more members by listing additional wallet addresses after Basemate's address.

Step 2: Send Welcome Message

xmtp conversation send-text <conversationId> "Welcome! 🔗"

Step 3: Wait for Indexing

Basemate indexes the group after detecting a message. Once indexed, tag writes will persist and the group appears in the discovery feed.

Step 4: Set Tags

curl -X PUT https://devconnectarg-production.up.railway.app/api/group/<groupId>/tags \
  -H "Content-Type: application/json" \
  -d '{"tags": ["defi", "trading", "base"]}'

The group is now live. Share with deeplinks: cbwallet://messaging/<conversationId>

Boost — Paid Discovery ($4.99/week)

  • +50 score boost in the trending algorithm
  • is_suggestable: true — agents recommend it, skips 5-member requirement
  • Top of discovery feed for 7 days

Agent Flow

  1. Transfer $4.99 USDC to the treasury address on Base Mainnet
  2. Call verify endpoint — POST /api/group/boost/verify-tx with { "groupId", "txHash" }

The backend verifies onchain: tx succeeded, USDC transfer to treasury, amount ≥ $4.99, no replay.

Auction — Bid for #1 Trending

Fully onchain smart contract on Base Mainnet. Winner's community gets the top trending slot.

Placing a Bid

  1. Approve USDC — USDC.approve(auctionContract, bidAmount)
  2. Place bid — placeBidSimple(bidAmount, maxAmount, communityName, description, "")

Reading Auction State
